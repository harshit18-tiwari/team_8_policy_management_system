# Backend

Node.js + Express backend with SQLite.

## commands
- npm install
- node dbInit.js (Run this once to setup DB)
- node server.js (Run this to start API)

## Endpoints
- POST /api/auth/login
- GET /api/products
- POST /api/premium/calculate
- POST /api/policies/purchase
- POST /api/payments/simulate
- POST /api/claims/file
